# Deployment

## Artifacts

The repo contains all of the pkg files, dev ids and passwords required for each build. The CI will automatically create the following, on each pr run, and for nightly builds.

 - zips for:
   - dev
   - qa
   - prod
 - signed packages for:
   - qa
   - prod
